# Code signing policy

## Поставщик подписи

Free code signing provided by [SignPath.io](https://signpath.io/), certificate
by [SignPath Foundation](https://signpath.org/).

После одобрения проекта службой SignPath эта подпись будет применяться к новым
официальным выпускам `CopyPhoto.exe`. Выпуск 0.1.0 создан до подключения
SignPath и поэтому подписи не содержит.

## Ответственные лица

- разработчик, коммитер и рецензент изменений —
  [Leonid Bolshakov](https://github.com/LeonidBolshakov);
- лицо, утверждающее запросы на подпись, —
  [Leonid Bolshakov](https://github.com/LeonidBolshakov).

Один человек совмещает эти роли, поскольку у проекта один сопровождающий.
Изменения сторонних участников принимаются только после проверки
сопровождающим.

## Сборка и утверждение подписи

Официальный EXE собирается из опубликованного исходного кода автоматическим
процессом GitHub Actions, описанным в
[`.github/workflows/build-windows.yml`](.github/workflows/build-windows.yml).
Сценарий выполняет автоматические тесты, статическую проверку типов и сборку
PyInstaller до публикации артефакта.

Каждый запрос на подпись должен быть отдельно и вручную утверждён лицом,
указанным выше. Подписывать разрешается только `CopyPhoto.exe`, полученный этим
автоматическим процессом из репозитория CopyPhoto.

## Конфиденциальность

Программа обрабатывает фотографии локально. Условия обработки данных приведены
в [политике конфиденциальности](PRIVACY.md).
